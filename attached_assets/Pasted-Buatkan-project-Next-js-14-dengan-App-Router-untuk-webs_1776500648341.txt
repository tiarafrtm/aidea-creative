Buatkan project Next.js 14 dengan App Router untuk website studio foto bernama "AideaCreative" yang berlokasi di Pujodadi, Pringsewu. Website ini adalah Smart Web E-Commerce yang menggabungkan sistem booking jasa foto dan toko produk foto online, dilengkapi fitur kecerdasan buatan (AI).

STACK YANG DIGUNAKAN:
- Framework: Next.js 14 (App Router)
- Database: Supabase (PostgreSQL)
- ORM: Prisma atau Supabase JS Client
- Styling: Tailwind CSS
- UI Component: shadcn/ui
- Auth: Supabase Auth
- Payment: Midtrans (mode sandbox)
- AI: Qwen API (qwen-turbo-latest)
- Hosting Target: Railway
- State Management: Zustand
- Form: React Hook Form + Zod

SETUP YANG DILAKUKAN:
1. Inisialisasi Next.js 14 dengan TypeScript
2. Install semua dependencies yang dibutuhkan
3. Setup Tailwind CSS dan shadcn/ui
4. Buat file .env.local dengan variabel berikut:
   - NEXT_PUBLIC_SUPABASE_URL=
   - NEXT_PUBLIC_SUPABASE_ANON_KEY=
   - SUPABASE_SERVICE_ROLE_KEY=
   - QWEN_API_KEY=
   - MIDTRANS_SERVER_KEY=
   - MIDTRANS_CLIENT_KEY=
   - NEXT_PUBLIC_APP_URL=
5. Buat file README.md dengan panduan setup
6. Buat konfigurasi Railway (railway.json dan Procfile)

Setelah selesai, tampilkan struktur folder yang telah dibuat.
